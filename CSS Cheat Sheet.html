CSS Selector Cheat Sheet

Declaration Function Example
"font-family" Changes font style of element font-family: rockwell;
"font-size" Changes font size of element, measured in pixels font-size: 18px;
"font-weight" Controls how bold or thin text appears. Values are "bold" and "normal" font-weight: bold;
"text-align" Aligns text to the element that holds it. Values are "center", "right", and "left" text-align: center;
"color" Styles an element's foreground (font) color color: azure;
"background-color" Styles an element's background color background-color: red;
"opacity" Changes the measure of how transparent or opaque an element is. Values range from 0 to 1, with 1 being fully visible and 0 being fully invisible opacity: 0.65;
"background-image" Sets an element's background to display an image background-image: url("./image.jpg");
"height" and "width" Used to change dimensions of elements. Value given in pixels height: 300px; width: 300px;
"list-style" Used to change bullet point style of list item elements list-style: square;
